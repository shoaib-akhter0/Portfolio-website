# Portfolio Website

A modern, responsive portfolio website showcasing AI engineering, data science, and full-stack development work.

## Features

- **Responsive Design** – Works seamlessly on desktop, tablet, and mobile
- **Project Showcase** – Display AI/ML, full-stack, and data science projects
- **Contact Section** – Easy ways to get in touch (email, WhatsApp, GitHub, Kaggle)
- **CV Download** – Direct CV download from the contact page
- **Smooth Navigation** – Dynamic page transitions and custom cursor effects

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize content with your information

## Files

- `index.html` – Main portfolio page
- `style.css` – Styling and animations
- `Update_cv.pdf` – CV file (update with your own)
